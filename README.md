## 💡 Проблема и решение

**Знакомая боль?**
Игры блокируются провайдером? Пинг скачет, а пакеты теряются? Ручная настройка `winws` (Zapret), проброс портов и анализ сетевых соединений через `netstat` отнимают часы, а любая ошибка приводит к сломанному интернету и необходимости вручную восстанавливать файлы?

**Решение, которое работает на вас**
**Fix Traffic Pipeline** — это умный инструмент с современным графическим интерфейсом, который берет всю сетевую рутину на себя. Он сам анализирует трафик целевых процессов, находит нужные IP и домены, генерирует правила для фаервола и DPI-обхода, а также бесшовно интегрирует Cloudflare WARP. Всё это — с гарантией безопасности благодаря системе автоматических бэкапов. Вы просто выбираете игру и нажимаете «Запустить».

## ⚡ Ключевые возможности

- 🎯 **Умный мониторинг трафика**: Автоматический перехват соединений целевых процессов с DNS-резолвингом и цветовой индикацией статусов в реальном времени (встроенный терминал).
- 🛡️ **Бесшовный обход DPI**: Интеграция с Zapret (`winws`). Автоматическая распаковка, генерация и применение правил на основе YAML-конфигов.
- 🌐 **Cloudflare WARP из коробки**: Автоматическая установка, подключение и настройка протокола MASQUE одним кликом.
- 💾 **Система безопасности (Backup & Restore)**: Автоматическое создание бэкапов папки `lists` перед изменениями. Случайная ошибка? Откат в один клик через диалог после остановки.
- 🐍 **Портативный Python**: Встроенное изолированное окружение Python 3.13. Не засоряет систему, не требует ручной установки зависимостей.
- 🎨 **Современный UI и темы**: Интерфейс на WPF с поддержкой кастомных Light/Dark тем, плавными анимациями и удобным управлением пресетами.
- ⚙️ **Гибкая конфигурация**: Поддержка пользовательских YAML-пресетов для любых игр (Apex Legends, Hunt: Showdown, Roblox и др.) с возможностью синхронизации из GitHub.
- 🛠️ **Сервисные утилиты**: Сброс DNS, очистка кэша, управление сетевыми адаптерами и файрволом Windows прямо из интерфейса.

## 🏁 Быстрый старт

### Для пользователей
1. Скачайте последнюю версию из раздела [Releases](https://github.com/AITISPEC/Fix-Traffic-Pipeline/releases).
2. Запустите `PlatformLauncher.exe` (потребуются права администратора).
3. Выберите игру из списка слева, нажмите **«Установить»** (для настройки правил фаервола), а затем **«Запустить фикс»**.

### Для разработчиков (Сборка из исходников)
Убедитесь, что у вас установлен [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).

```bash
# Клонируйте репозиторий
git clone https://github.com/AITISPEC/Fix-Traffic-Pipeline.git
cd Fix-Traffic-Pipeline

# Сборка проекта
dotnet build PlatformLauncher/PlatformLauncher.csproj -c Release -r win-x64

# Запуск
./PlatformLauncher/bin/x64/Release/net8.0-windows/PlatformLauncher.exe
```
